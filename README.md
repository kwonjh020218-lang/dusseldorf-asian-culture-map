# 🇩🇪🎌 뒤셀도르프 아시안 컬처 맵

뒤셀도르프의 한인·아시안 맛집, 마트, 만화카페, 서점, 명소, 굿즈샵 등 **아시아 문화를 즐길 수 있는 122곳**을 지도에서 찾아보고, 나만의 코스를 짤 수 있는 웹앱입니다.

**🔗 라이브 데모: [바로 열어보기](https://kwonjh020218-lang.github.io/dusseldorf-asian-culture-map/)**

## 왜 만들었나요

뒤셀도르프에는 유럽 최대 규모의 일본인 커뮤니티가 있는 "리틀 도쿄"(임머만 거리)가 있어서, 한식·일식·중식·베트남·태국 음식점부터 만화카페, 일본 서점, K-pop 굿즈샵까지 다양한 아시아 문화 스팟이 밀집해 있습니다. 하지만 이 정보를 한눈에 볼 수 있는 곳이 없어서 직접 지도를 만들었습니다.

바이브 코딩(AI 페어 프로그래밍)으로 기획부터 배포까지 진행하며, 실제 사용자 관점에서 필요한 기능을 계속 다듬어나간 프로젝트입니다.

## 주요 기능

- 🔍 **검색 & 필터**: 이름 검색, 13개 카테고리(국가별 음식 + 마트/카페/오락/서점/디저트/명소/굿즈), 가격대, 비건/매운맛/포토스팟 여부
- 📍 **실시간 위치 기반**: 내 위치에서 거리순 정렬, 근처 카페·디저트 자동 추천
- 🧭 **나만의 코스 빌더**: 팝업에서 코스에 담고 순서 조정, 담을 때마다 자동으로 도보 경로가 그려짐 (내 위치 또는 뒤셀도르프 중앙역 기준)
- 🎲 **추천 코스**: 필터 조건을 존중하며 "식당 → 카페/디저트 → 놀거리"를 자동으로 골라주는 기능
- ⭐ **개인화**: 즐겨찾기, 방문 체크, 방문 수에 따른 레벨 뱃지 (localStorage 저장)
- 🗺️ **지도 UX**: 마커 클러스터링, 호버 툴팁, 클릭 시 마커 강조 애니메이션
- 🎲 **오늘 뭐 하지?**: 현재 필터 기준으로 랜덤 추천해주는 룰렛

## 기술 스택

- **순수 HTML/CSS/JavaScript** — 프레임워크·번들러 없이 바닐라 JS로 구현
- **[Leaflet.js](https://leafletjs.com/)** — 지도 렌더링
- **[Leaflet.markercluster](https://github.com/Leaflet/Leaflet.markercluster)** — 마커 클러스터링
- **[Leaflet Routing Machine](http://www.liedman.net/leaflet-routing-machine/)** + OSRM — 경로 안내
- **Google Places API** (개발 단계에서만) — 실제 장소 데이터(좌표, 카테고리) 수집에 활용, 배포된 앱 자체는 API 키를 쓰지 않음
- **GitHub Pages** — 무료 정적 호스팅

CDN 라이브러리는 unpkg → jsdelivr → cdnjs 순서로 자동 재시도하도록 만들어서, 특정 CDN이 네트워크 환경에 따라 차단되어도 앱이 죽지 않도록 방어적으로 설계했습니다.

## 로컬에서 실행하기

별도 설치 과정 없이 `index.html`을 브라우저로 열면 바로 실행됩니다. (지도 타일과 라이브러리는 CDN에서 불러오므로 인터넷 연결이 필요합니다.)

```bash
git clone https://github.com/kwonjh020218-lang/dusseldorf-asian-culture-map.git
cd dusseldorf-asian-culture-map
# index.html을 브라우저로 열기
```

## 알고 있는 한계 / 앞으로 개선하고 싶은 점

실무 프로덕션 코드였다면 아래처럼 진행했을 것입니다 (지금은 단일 파일 프로젝트 특성상 의도적으로 타협한 부분들입니다):

- **상태 관리**: 지금은 모듈 스코프의 전역 변수로 상태를 관리합니다. React 등으로 옮긴다면 컴포넌트 상태/컨텍스트로 정리하고 싶습니다.
- **모듈 분리**: `script.js` 하나에 모든 로직이 들어있습니다. 기능별로 파일을 나누고 번들러(Vite 등)를 도입하면 유지보수가 쉬워질 것입니다.
- **접근성**: 주요 버튼에 `aria-label`을 추가했지만, 스크린 리더 전체 흐름 테스트는 아직 못 했습니다.
- **데이터 정확도**: 122곳의 좌표는 Google Places API 기반이라 정확하지만, 비건/매운맛 여부·가격대·대표 메뉴 일부는 리뷰 기반 추정치입니다. 직접 방문 검증한 곳부터 순차적으로 업데이트할 계획입니다.
